## ✨ Features

| Feature | Description |
|---|---|
| 🏁 **FFlag Editor** | Search, toggle, and persist fast flags with ease |
| 🎨 **Themes & Appearance** | Curated presets plus fine-grained visual controls |

## 🚀 Quick Start

**1. Download** the latest release from the [Releases page](https://github.com/DTstrap/DTstrap/releases).

**2. Run** `DTstrap.exe` — it will detect Roblox and configure itself automatically.

**3. Customize** via the Flags, Themes, or Mods tabs. Create a Profile to save your full setup.

### 🔒 Optional Hardening

Open **Settings → Integrity** to:
- *Lock my flags* — prevent accidental edits.
- *Validate install on launch* — hash-check core files on every startup.
